== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Affiliate Links > Add New** to start adding links
4. Use `/go/your-slug` on your website to redirect users
5. Check **Click Reports** for analytics
